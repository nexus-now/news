AEGIS — Verification & Copyright
EVERY STORY VERIFIED AND PROTECTED
✅
VERIFIED
Core claims confirmed from 2+ independent sources. High confidence. Published normally.
⚡
DEVELOPING
Story confirmed but details still emerging. Published with "developing story" caveat.
🔍
UNVERIFIED
Single source only. Published with clear "unverified" disclaimer embedded in all content.
⛔
BLOCKED
Contradicting sources, impossible claims, or ethical issues. Story is blocked — never published.
# Copyright ID format: NN-YYYY-MMDD-HHMMSS-CATEGORYCODE-HASH Example IDs: NN-2026-0404-143022-TEC-A7F3 ← Technology story NN-2026-0404-180055-BIZ-C2D9 ← Business story NN-2026-0404-180111-SPT-F1A8 ← Sports story # This ID appears in: YouTube description footer + JSON metadata Instagram caption last line of every post Tweet thread final tweet Video script spoken at end of broadcast Website article JSON-LD structured data Google Sheets copyright registry log

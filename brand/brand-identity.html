Brand Usage Rules
✅ DO
  • Use logo-dark.png on all dark backgrounds
  • Use logo-light.png on white/light backgrounds
  • Keep minimum clear space = height of the "N" around the logo
  • Use Signal Red (#E81A1A) as the only accent colour
  • Use Bebas Neue for all headlines and display text
  • Place watermark in bottom-right of all videos
  • Include copyright ID in all published content
  • Use icon-512.png for all profile pictures
❌ DON'T
  • Never stretch or distort the logo
  • Never change the logo colours
  • Never place logo on busy photo backgrounds without overlay
  • Never use other red shades — only #E81A1A
  • Never use Arial, Roboto, or system fonts for brand text
  • Never remove the signal icon from the logo
  • Never publish content without AEGIS copyright ID
  • Never use the logo smaller than 120px wide
